1. JSP
- Java Server Pages
- used to create web pages
- server side
- used for presentation UI
- JSP -> converted to servlet -> then compiled 
- slow performance
- easy to write and maintain
- slower excecution
- HTMl + Java

2. Servlet
- used to create web application
- server side
- used for business logic
- already compiled
- faster performance
- complex
- faster excecution
- Pure Java

3. Applet
- used for GUI and graphics
- client side
- runs in browser
- outdated
